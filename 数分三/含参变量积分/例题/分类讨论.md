# 例题  
设 $F(y) = \int_{a}^{b} f(x) |y - x| dx \quad (a < b)$，其中 $f(x)$ 为可微函数，求 $F''(y)$。

# 解答  
## 方法分析  
本题涉及含绝对值函数的积分，需要根据 $y$ 与区间 $[a,b]$ 的位置关系分段讨论。绝对值 $|y-x|$ 在 $x=y$ 处不可导，因此需将积分区间拆分为 $[a,y]$ 和 $[y,b]$ 分别处理，再对 $F(y)$ 求二阶导数。

## 解题步骤  
### Step 1：分情况讨论 $F(y)$ 的表达式  
- 当 $y \le a$ 时，$|y-x| = x-y$，  
  $$F(y) = \int_{a}^{b} f(x)(x-y) dx.$$  
- 当 $y \ge b$ 时，$|y-x| = y-x$，  
  $$F(y) = \int_{a}^{b} f(x)(y-x) dx.$$  
- 当 $a < y < b$ 时，需拆分积分区间：  
  $$F(y) = \int_{a}^{y} f(x)(y-x) dx + \int_{y}^{b} f(x)(x-y) dx.$$

### Step 2：求一阶导数 $F'(y)$  
- 当 $y \le a$ 时，  
  $$F'(y) = -\int_{a}^{b} f(x) dx.$$  
- 当 $y \ge b$ 时，  
  $$F'(y) = \int_{a}^{b} f(x) dx.$$  
- 当 $a < y < b$ 时，利用含参变量积分的求导法则（莱布尼茨公式）：  
  $$F'(y) = \int_{a}^{y} f(x) dx - \int_{y}^{b} f(x) dx.$$

### Step 3：求二阶导数 $F''(y)$  
- 当 $y \le a$ 或 $y \ge b$ 时，$F'(y)$ 为常数，  
  $$F''(y) = 0.$$  
- 当 $a < y < b$ 时，对 $F'(y)$ 再求导：  
  $$F''(y) = f(y) - (-f(y)) = 2f(y).$$

**最终答案：**  
\[
\boxed{F''(y) = \begin{cases} 
0, & y \le a \text{ 或 } y \ge b, \\ 
2f(y), & a < y < b.
\end{cases}}
\]

# 核心技巧  
## 分段处理绝对值  
- 根据 $y$ 与积分区间 $[a,b]$ 的关系，将 $|y-x|$ 写为分段形式，避免直接对绝对值求导。

## 含参变量积分求导  
- 当积分限含参变量 $y$ 时，使用莱布尼茨公式：  
  $$\frac{d}{dy} \int_{\alpha(y)}^{\beta(y)} g(x,y) dx = \int_{\alpha(y)}^{\beta(y)} \frac{\partial g}{\partial y} dx + g(\beta(y),y) \beta'(y) - g(\alpha(y),y) \alpha'(y).$$

# 经验总结  
1. **分段讨论关键点**：绝对值函数在 $x=y$ 处不可导，必须拆分积分区间。  
2. **边界情况处理**：在区间外部（$y \le a$ 或 $y \ge b$）时，$F(y)$ 为线性函数，二阶导数为零。  
3. **含参积分求导**：注意积分上下限含参变量时，需应用莱布尼茨公式，并正确计算边界项贡献。
