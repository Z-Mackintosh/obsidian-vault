### 预习核心概念

为了掌握有界闭区域上的重积分，你需要理解以下核心概念：

1.  **二重积分的定义**
    *   理解如何通过分割区域、取点、作黎曼和、然后取极限来定义二重积分。
    *   了解从矩形区域到更一般有界闭区域的推广。

2.  **二重积分的重要性质**
    *   **线性性质**：常数可以提出，和的积分等于积分的和。
    *   **区域可加性**：如果一个区域可以被分成两个子区域，那么在整个区域上的积分等于在两个子区域上积分的和。
    *   **保号性与比较定理**：如果一个函数在区域上非负，其积分也非负。如果一个函数大于另一个函数，其积分也相应更大。
    *   **估值定理与中值定理**：用于估计积分的范围。

3.  **二重积分的计算方法**
    *   **傅比尼定理 (Fubini's Theorem)**：核心中的核心。它允许我们将二重积分转化为两次定积分（即**累次积分**）来计算。
    *   **在直角坐标系下计算**：
        *   **X-型区域**：积分区域由 `a ≤ x ≤ b`, `g(x) ≤ y ≤ h(x)` 描述，积分次序为 `dy dx`。
        *   **Y-型区域**：积分区域由 `c ≤ y ≤ d`, `p(y) ≤ x ≤ q(y)` 描述，积分次序为 `dx dy`。
        *   掌握如何根据区域形状选择合适的积分次序，以及如何在两种次序之间进行**交换**。
    *   **在极坐标系下计算**：
        *   理解坐标变换和雅可比行列式（Jacobian），知道面积微元 `dx dy` 变为 `r dr dθ`。
        *   学会如何用极坐标确定积分区域的边界，从而写出正确的积分限。

4.  **二重积分的几何应用**
    *   计算以积分区域为底，以 `z = f(x, y)` 为顶的**曲顶柱体的体积**。
    *   当被积函数为 `f(x, y) = 1` 时，二重积分的值就是**积分区域的面积**。
