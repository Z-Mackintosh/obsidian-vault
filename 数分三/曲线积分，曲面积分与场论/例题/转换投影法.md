# 题目
$\iint_{\Sigma} [f(x, y, z) + x] dy dz + [2f(x, y, z) + y] dz dx + [f(x, y, z) + z] dx dy$其中$f (x, y, z)$ 为连续函数，$\Sigma$是平面$x - y + z = 1$ 在第四卦限部分的上侧。
# 解答
## 两类曲面关系
1. 平面 $F (x, y, z) = x - y + z - 1 = 0$ 上的单位法向量为
$$n = \frac{1}{\sqrt{F_x'^2 + F_y'^2 + F_z'^2}} (F_x', F_y', F_z') = \frac{1}{\sqrt{3}} (1, -1, 1) \quad (z > 0 \text{（取上侧）})$$
其方向余弦为 $\cos \alpha = \frac{1}{\sqrt{3}}$, $\cos \beta = -\frac{1}{\sqrt{3}}$, $\cos \gamma = \frac{1}{\sqrt{3}}$。由两类曲面积分的关系得
$$\iint_{\Sigma} [f(x, y, z) + x] \, dy \, dz + [2f(x, y, z) + y] \, dz \, dx + [f(x, y, z) + z] \, dx \, dy$$
$$= \iint_{\Sigma} [(f + x) \cos \alpha + (2f + y) \cos \beta + (f + z) \cos \gamma] \, dS$$
$$= \frac{1}{\sqrt{3}} \iint_{\Sigma} [(f + x) - (2f + y) + (f + z)] \, dS$$
$$= \frac{1}{\sqrt{3}} \iint_{\Sigma} (x - y + z) \, dS \quad \text{代入平面方程} \quad \frac{1}{\sqrt{3}} \iint_{\Sigma} dS = \frac{1}{\sqrt{3}} \times \frac{\sqrt{3}}{2} = \frac{1}{2}$$
## 直接投影
## 角度二)直接用转换投影法公式
$$
z = -x + y + 1, z_x' = -1, z_y' = 1
$$
$$
\iint_{\Sigma} \left[ f(x, y, z) + x \right] dy dz + \left[ 2 f(x, y, z) + y \right] dz dx + \left[ f(x, y, z) + z \right] dx dy
$$
$$
= \iint_{\Sigma} \left\{ \left[ f(x, y, z) + x \right] (-z_x') + \left[ 2 f(x, y, z) + y \right] (-z_y') + \left[ f(x, y, z) + z \right] \right\} dx dy
$$
$$
= \iint_{\Sigma} \left\{ \left[ f(x, y, z) + x \right] - \left[ 2 f(x, y, z) + y \right] + \left[ f(x, y, z) + z \right] \right\} dx dy
$$
$$
= \iint_{D_{xy}} (x - y + z) dx dy = \iint_{D_{xy}} (x - y + (-x + y + 1)) dx dy = \iint_{D_{xy}} dx dy = \frac{1}{2}
$$
>  注意让 $z$ 为 0, 那么方程化为——或者说投影面为 $y=x-1$
