# 例题
计算曲线积分
$$ \int\limits_{L}\mid x\mid^{\frac{1}{3}}\mathrm{d}s,\text{其中 }L\text{ 为星形线 }x^{\frac{2}{3}}+y^{\frac{2}{3}}=a^{\frac{2}{3}};$$

# 解答
## 换元
令 $x = a\cos^3t, y = a\sin^3t$
## 转化
根据[[面积，弧长，体积的积分计算]]，转化 $ds$
对于**参数方程**
第一类曲线积分的计算公式需要用到弧长微分 `ds`。`ds` 的计算公式为： `ds = sqrt([x'(t)]² + [y'(t)]²) dt`

我们来计算 `x(t)` 和 `y(t)` 对 `t` 的导数：

- `x'(t) = d/dt (a * cos³(t)) = a * 3cos²(t) * (-sin(t)) = -3a * sin(t) * cos²(t)`
- `y'(t) = d/dt (a * sin³(t)) = a * 3sin²(t) * cos(t) = 3a * cos(t) * sin²(t)`

接下来，计算 `[x'(t)]² + [y'(t)]²`：

- `(-3a * sin(t) * cos²(t))² + (3a * cos(t) * sin²(t))²`
- `= 9a² * sin²(t) * cos⁴(t) + 9a² * cos²(t) * sin⁴(t)`
- `= 9a² * sin²(t) * cos²(t) * (cos²(t) + sin²(t))`
- `= 9a² * sin²(t) * cos²(t)` （因为 `cos²(t) + sin²(t) = 1`）

最后，代入 `ds` 的公式：

- `ds = sqrt(9a² * sin²(t) * cos²(t)) dt = |3a * sin(t) * cos(t)| dt`
- 由于 `a` 通常是正常数，所以 `ds = 3a * |sin(t)cos(t)| dt`。
$ds = 3a|\sin t\cos t|dt$, 

## 计算
$\int_L |x|^{\frac{1}{3}}ds = 3a^{\frac{4}{3}}\int_0^{2\pi} |\sin t\cos^2t|dt = 12a^{\frac{4}{3}}\int_0^{\frac{\pi}{2}} \sin t\cos^2tdt = 4a^{\frac{4}{3}}$
