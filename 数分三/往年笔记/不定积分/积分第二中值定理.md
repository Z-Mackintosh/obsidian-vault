# 引理
$f(x)$ 在 $[a,b]$ 上单调且非负$g(x)$ 在 $[a,b]$ 上可积
$\Rightarrow \exists \xi \in [a,b]:\int_a^b f(x)g(x)\,dx = f(a) \int_a^\xi g(x)\,dx$

# 积分第二中值定理
## 定理
$f(x) \text{在 } [a,b] \text{上可积}g(x) \text{在 } [a,b] \text{上单调}$
$\Rightarrow \exists \xi \in [a,b]:\int_a^b f(x)g(x)\,dx = g(a) \int_a^\xi f(x)\,dx + g(b) \int_\xi^b f(x)\,dx$
## 结论
(1). 若 $g(x)$ 在 $[a,b]$ 上单调增且 $g(a) \geq 0$，有 $\exists \xi \in [a,b]$：  
$$
\int_a^b f(x)g(x)\,dx = g(b) \int_\xi^b f(x)\,dx
$$

(2). 若 $g(x)$ 在 $[a,b]$ 上单调减且 $g(b) \geq 0$，有 $\exists \xi \in [a,b]$：  
$$
\int_a^b f(x)g(x)\,dx = g(a) \int_a^\xi f(x)\,dx
$$
# A-D判别法
## 无穷限
若下列两条件之一满足，则  
$$
\int_a^{+\infty} f(x)g(x)\,dx \text{收敛}.
$$

(1). (Abel 判别法)  
$$
\int_a^{+\infty} f(x)\,dx \text{收敛}, \, g(x) \text{在 } [a, +\infty) \text{上单调有界};
$$

(2). (Dirichlet 判别法)  
$$
F(A) = \int_a^A f(x)\,dx \text{在 } [a, +\infty) \text{上有界}, \, g(x) \text{在 } [a, +\infty) \text{上单调}, \text{且 } \lim_{x \to +\infty} g(x) = 0.
$$

## 瑕点
若下列两条件之一满足，则  
$$
\int_a^b f(x)g(x)\,dx \text{收敛} \quad (b \text{为唯一瑕点})。
$$

(1). (Abel 判别法)  
$$
\int_a^b f(x)\,dx \text{收敛}, \, g(x) \text{在 } [a,b] \text{上单调有界};
$$

(2). (Dirichlet 判别法)  
$$
F(\eta) = \int_a^{b-\eta} f(x)\,dx \text{在 } (0, b-a] \text{上有界}, \, g(x) \text{在 } [a,b) \text{上单调}, \text{且 } \lim_{x \to b^-} g(x) = 0.
$$
