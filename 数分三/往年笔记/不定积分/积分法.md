#  换元积分法
## 第一类
**开始不理解的点**：[[不定积分#^f8t06hfaedj|(cos  s)  「dx ]]
似乎是$u=g(x),g^{'}(x)dx=du$

## 第二类
$x=g(t),同时微分，dx=g^{'}(t)dt$
适用于**三角换元**
被积函数含 $\sqrt{a^2 - x^2} (a > 0)$，令 $x = a \sin t$，$t \in \left[ -\frac{\pi}{2}, \frac{\pi}{2} \right]$，则 $\sqrt{a^2 - x^2} = a \cos t$；

被积函数含 $\sqrt{x^2 + a^2} (a > 0)$，令 $x = a \tan t$，$t \in \left( -\frac{\pi}{2}, \frac{\pi}{2} \right)$，则 $\sqrt{x^2 + a^2} = a \sec t$；

被积函数含 $\sqrt{x^2 - a^2} (a > 0)$，令 $x = a \sec t$，$t \in \left[ 0, \frac{\pi}{2} \right) \cup \left( \frac{\pi}{2}, \pi \right]$，则 $\sqrt{x^2 - a^2} = a \tan t$；

# 分部积分法
## 公式
$$
\int u(x) v'(x) \, dx = u(x) v(x) - \int v(x) u'(x) \, dx
$$
$$
\int u(x)  d(g(x)) = u(x) g(x) - \int g(x) u'(x) \, dx
$$
## 技巧
选取u及v'的一般方法:

把被积函数视为两个函数之积，按“==反对幂指三==”的顺序，前者为$u$后者为$v'$.

例4. 求$∫arccos x dx.$

解: 令$u = arccos x, v' = 1$, 则
$u' = -\frac{1}{\sqrt{ 1-x^{2}} },  v = x$
 原式 =$x arccos x + ∫ \frac{x}{\sqrt{ 1-x^{2} }}dx$
= $x arccos x - \frac{1}{2} ∫(1-x^2)^{-1/2} d(1-x^{2})$
= $x arccos x -\sqrt{ 1-x^{2} } + C$

## 题型
1. 直接分部化简积分；
2. 分部产生循环式，由此解出积分式；
   (注意: 两次分部选择的 $u, v$ 函数类型不变，解出积分后加 $C$)
   求 $\int e^x \sin x \, dx$.
解：令 $u = \sin x$, $v' = e^x$, 则
$u' = \cos x, \quad v = e^x$
$\therefore$ 原式 $= e^x \sin x - \int e^x \cos x \, dx$
再令 $u = \cos x$, $v' = e^x$, 则
$u' = -\sin x, \quad v = e^x$
$原式= e^x \sin x - e^x \cos x - \int e^x \sin x \, dx$
故 原式 $= \frac{1}{2} e^x (\sin x - \cos x) + C$
说明：也可设 $u = e^x$, $v'$ 为三角函数，但两次所设类型必须一致.
3. 对含自然数 $n$ 的积分，通过分部积分建立递推公式。

# 三角函数有关技巧
## 结论
含 $\sin^m x \cdot \cos^n x$ 的积分

- $\sin x$ 奇次，$\cos x$ 偶次 $\rightarrow d(\cos x)$ 或 $d(\sec x)$
- $\cos x$ 奇次，$\sin x$ 偶次 $\rightarrow d(\sin x)$ 或 $d(\csc x)$
- $\cos x$ 和 $\sin x$ 同为偶次或奇次 $\rightarrow d(\tan x)$ 或 $d(\cot x)$

## 公式
万能代换公式，令 $u = \tan \frac{x}{2}$，则 $\sin x = \frac{2u}{1+u^2}$，$\cos x = \frac{1-u^2}{1+u^2}$，$dx = \frac{2}{1+u^2} du$，从而

$$
\int R(\sin x, \cos x) dx = \int R\left(\frac{2u}{1+u^2}, \frac{1-u^2}{1+u^2}\right) \frac{2}{1+u^2} du
$$


化为有理函数积分。
