# 寡头垄断 - 课堂纪要

## 第1讲：寡头垄断概述

### 1.1 定义与特征
**教师讲解**：
- 寡头垄断：市场上只有少数几个厂商
- 关键特征：策略性相互依赖
- 每个厂商的决策都会影响市场价格

**课堂讨论**：
- 现实例子：汽车行业、航空公司、电信运营商
- 与完全竞争、垄断的区别

### 1.2 分析框架
**教师强调**：
- 需要预测竞争对手的反应
- 博弈论的应用
- 时序问题：同时决策 vs 序贯决策

## 第2讲：产量领导模型（斯塔克尔伯格）

### 2.1 模型设定
**教师板书**：
- 两个厂商：领导者（厂商1）和追随者（厂商2）
- 领导者先行动，追随者后行动
- 市场需求：$P = a - bQ$

### 2.2 追随者决策
**推导过程**：
追随者利润函数：
$$\pi_2 = [a - b(q_1 + q_2)]q_2 - C_2(q_2)$$

一阶条件：
$$\frac{\partial \pi_2}{\partial q_2} = a - bq_1 - 2bq_2 - MC_2 = 0$$

得到反应函数：
$$q_2 = \frac{a - MC_2}{2b} - \frac{1}{2}q_1$$

### 2.3 领导者决策
**教师讲解**：
领导者知道追随者的反应函数，因此：
$$\pi_1 = [a - b(q_1 + f_2(q_1))]q_1 - C_1(q_1)$$

**学生提问**：为什么领导者有优势？
**教师回答**：因为领导者能影响追随者的决策

## 第3讲：古诺模型

### 3.1 同时决策
**教师讲解**：
- 两个厂商同时选择产量
- 每个厂商预测对方的产量
- 均衡时预测正确

### 3.2 反应函数
**推导**：
厂商1的利润：
$$\pi_1 = [a - b(q_1 + q_2)]q_1 - cq_1$$

一阶条件：
$$a - 2bq_1 - bq_2 - c = 0$$

得到：
$$q_1 = \frac{a - c}{2b} - \frac{1}{2}q_2$$

同理：
$$q_2 = \frac{a - c}{2b} - \frac{1}{2}q_1$$

### 3.3 古诺均衡
**求解**：
联立两个反应函数：
$$q_1^* = q_2^* = \frac{a - c}{3b}$$

**课堂练习**：计算均衡价格和利润

## 第4讲：伯特兰模型

### 4.1 价格竞争
**教师讲解**：
- 厂商同时选择价格
- 消费者购买价格最低的产品
- 产品同质

### 4.2 伯特兰悖论
**推导**：
如果 $P_1 > P_2$，厂商1销量为0
如果 $P_1 = P_2 > c$，厂商有动机降价
均衡：$P_1^* = P_2^* = c$

**学生困惑**：为什么两个厂商就能达到完全竞争？
**教师解释**：这是模型的极端假设导致的

### 4.3 解决悖论
**教师讲解**：
1. 产能约束
2. 产品差异化
3. 重复博弈

## 第5讲：卡特尔与串谋

### 5.1 卡特尔形成
**教师讲解**：
- 厂商联合行动
- 像垄断者一样决策
- 分配产量配额

### 5.2 最优决策
**推导**：
总利润最大化：
$$\max_{q_1,q_2} [a - b(q_1 + q_2)](q_1 + q_2) - c(q_1 + q_2)$$

一阶条件：
$$MR = MC_1 = MC_2$$

### 5.3 欺骗动机
**教师强调**：
每个厂商都有动机生产超过配额
欺骗利润：$\pi^{cheat} > \pi^{cartel}$

**课堂讨论**：为什么现实中卡特尔难以维持？

## 第6讲：重复博弈与惩罚

### 6.1 冷酷策略
**教师讲解**：
- 开始合作
- 如果欺骗，永远惩罚
- 惩罚阶段回到古诺均衡

### 6.2 合作条件
**推导**：
合作可持续当：
$$\frac{\pi^{coop}}{1-\delta} \geq \pi^{cheat} + \frac{\delta}{1-\delta}\pi^{punish}$$

**教师解释**：$\delta$ 是贴现因子，反映对未来的重视程度

### 6.3 影响因素
**课堂讨论**：
- 检测欺骗的难度
- 惩罚的严厉程度
- 厂商数量

## 第7讲：模型比较与应用

### 7.1 模型总结
**教师总结表格**：

| 模型 | 决策变量 | 时序 | 均衡特点 |
|------|----------|------|----------|
| 斯塔克尔伯格 | 产量 | 序贯 | 领导者优势 |
| 古诺 | 产量 | 同时 | 介于垄断竞争 |
| 伯特兰 | 价格 | 同时 | 完全竞争 |
| 卡特尔 | 产量 | 合作 | 垄断结果 |

### 7.2 现实应用
**教师举例**：
- OPEC：卡特尔
- 英特尔-AMD：古诺竞争
- 微软：斯塔克尔伯格领导者

### 7.3 扩展讨论
**学生提问**：产品差异化如何处理？
**教师回答**：使用霍特林模型，考虑产品空间位置

## 课堂重点回顾

### 关键概念
1. **策略性相互依赖**：寡头垄断的核心特征
2. **反应函数**：描述厂商如何应对竞争对手
3. **均衡**：所有厂商都没有动机改变决策的状态
4. **先动优势**：斯塔克尔伯格模型中的领导者优势

### 重要结论
1. 古诺均衡产量高于垄断，低于完全竞争
2. 伯特兰模型在同质产品下导致完全竞争结果
3. 卡特尔不稳定，存在欺骗动机
4. 重复博弈可以支持合作

### 学习方法建议
1. 掌握每个模型的假设条件
2. 理解反应函数的推导
3. 比较不同模型的均衡结果
4. 联系现实世界的寡头垄断案例
