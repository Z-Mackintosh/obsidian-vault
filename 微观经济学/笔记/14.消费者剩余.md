# 交易福利的货币价值的测量方法
- 消费者剩余的变化
- 将等价变化：效用水平相等
- 补偿变化：效用水平相等
# [[消费者剩余]]
- 使用需求曲线代替价格保留曲线
	- 二者在拟线性效应下重合 (零收入效应)
	- 大部分情况下价格保留曲线不易求
	- 区别：
		- 保留价格曲线有序地描述了连续单个商品的价值
		- 一般需求曲线描述了消费者同时购买q单位商品时愿意支付的最高价格
# 补偿变化
1. 某一种商品的价格发生变化，预算线对应端点移动
2. 新预算线和某一个无差异曲线相切
3. 平移原预算线与该无差异曲线相切
![[14.消费者剩余 2025-09-24 16.39.48.excalidraw]]
>  蓝色新预算线与绿色原预算线之间的差异如果得到补偿，恰好满足消费者。即这部分的补充为消看费者福利

# 等价变化
1. 某一种商品价格变化，预算线对应端点移动
2. 新预算线向下平移，与原预算线切于同一条无差异曲线
![[14.消费者剩余 2025-09-24 16.46.47.excalidraw]]
两条预算线之间的差值，表示从消费者处拿走这些差值可以将消费者效应还原，差值即剩余
