# 短期行业供给
行业供给或者说市场供给曲线就是厂商供给之和。
# 短期行业均衡
短期内，只要价格和产量的组合高于平均可变成本曲线，那么即使是亏损的，维持经营也要好于停产。
# 长期行业均衡
对于自由进入行业，厂商自由进入，价格不低于长期平均成本。
# 长期供给曲线
对于上述结论，我们还可以得出
> 在一个可以自由进入的竞争行业，不可能获得高于零利润太多的利润。

# 不变要素和经济租金
- 存在某一些行业，厂商的数目不变
    如资源开采，存在执照和许可证的行业
> 但是，仍然存在着一种力量，使得利润趋近于0 。

注意，我们按照市场价格即**机会成本**来确定每一种生产要素的价格。
假设：
1. 你对一片土地的投资使你每年盈利$x$美元
2. 那么你每年愿意为这片土地投资而愿意放弃的其他机会成本是多少
3. 是$x$美元。
# 经济组金
定义：支付给生产要素的报酬超过为获得该要素而必须支付的最低报酬的部分。
- 租金和生产者剩余是同一个概念
- 均衡价格决定租金，而不是租金决定均衡价格
- 厂商沿着边际成本曲线提供产品，边际成本与不变要素的支出无关
- 租金将调整到利润等于0为止。
