# 投入与产出 
- 生产的投入称为生产要素
- 资本物品
	- 金融资本
	- 物质资本
>  我们不会像政治经济学一样考虑投入和产出的类别，而是研究数量。

# 描述技术约束
- **生产集**：由技术约束导致，只有某些投入组合才有可能产出既定的产量。
	- 产出与投入的生产集函数
- 如果投入有两种，那么生产集函数就是二元函数，同时有类似与无差异曲线的概念：等产线。
# 技术的例子
- 固定比例：施工中人与铲子的两种投入。一个人只能使用一个铲子。
	类似于无差异中的完全互补
- 完全替代：做作业时的铅笔种类：无所谓。红铅笔和蓝铅笔相互完全替代
- 柯布-道格拉斯生产函数：其与柯布道格拉斯无差异曲线拥有相同的良好性状。
	$$ y = f(x_1, x_2) = A x_1^\alpha x_2^\beta $$
	• $A$ 是技术系数（总要素生产率）
	• $\alpha$ 和 $\beta$ 是产出弹性系数 
# 技术的特征
1. 投入就有产出；多余产出无损失
2. 技术是**凸**的
# 边际产品
对于等产量线上一点 $(x_{1},x_{2})$，现在考虑增加要素 1 使用量。
$$ \frac{\Delta y}{\Delta x_1}=\frac{f(x_1+\Delta x_1,\:x_2)-f(x_1,\:x_2)}{\Delta x_1}$$
该比率称为要素一的边际产品。
> 边际产品是一种比率，描述相对于每一单位额外投入的额外产量。

# 技术替代率
类似与边际替代率，称为 $TRS$，即放弃一点要素一下，投入多少要素二使得产出不变。
$$ \text{TRS }(x_1,\:x_2)=\frac{\Delta x_2}{\Delta x_1}=-\frac{MP_1(x_1,\:x_2)}{MP_2(x_1,\:x_2)}$$
# 边际产品递减
增加一个要素的投入，显然在单调性成立的前提下，总产量会增加，但是是一一种比率递减的情况下增加。

# 技术替代率递减
增加要素一同时调整要素二以保证产量不变时，技术替代率会变小。

# 长期与短期
经济学家对长期和短期所作的区分如下：在短期内某些生产要素是固定的，譬如固定的土地数量、固定的工厂规模、固定的机器设备等等；在长期内所有生产要素都是可以变动的。

# 规模报酬
- 当把所有投入成比例增加，通常情况下产出也相同比例增加
- 注意，一种生产技术既显示出规模报酬不变又显示出每种生产要素的边际产品递减是完全可能的。因为规模报酬阐述的是增加所有投入时产量将怎样变动，而边际产品递减指的是增加一种投入，而使其他投入固定不变时产量将怎样变化。
- 规模报酬递增&规模报酬递减


对于柯布-道格拉斯生产函数：
- 当 $\alpha + \beta = 1$ 时，规模报酬不变
- 当 $\alpha + \beta > 1$ 时，规模报酬递增
- 当 $\alpha + \beta < 1$ 时，规模报酬递减

# 要素的边际生产与规模报酬的关系

## 边际产品与规模报酬的区分
- **边际产品递减**：指在保持其他要素不变时，增加某一要素投入，其边际产量递减
- **规模报酬**：指所有要素按相同比例变化时，产出的变化比例

## 两者可以同时存在
- 一个生产函数可以同时具有：
  - 边际产品递减（短期现象）
  - 规模报酬不变/递增/递减（长期现象）

## 数学关系
对于柯布-道格拉斯生产函数 $y = A x_1^\alpha x_2^\beta$：
- 要素1的边际产品：$MP_1 = \alpha A x_1^{\alpha-1} x_2^\beta$
- 要素2的边际产品：$MP_2 = \beta A x_1^\alpha x_2^{\beta-1}$
- 当 $\alpha < 1$ 时，要素1的边际产品递减
- 当 $\beta < 1$ 时，要素2的边际产品递减
- 规模报酬由 $\alpha + \beta$ 决定

## 关键结论
- 边际产品递减是**短期**特征（部分要素固定）
- 规模报酬是**长期**特征（所有要素可变）
- 两者描述不同维度的生产技术特征
- 边际产品递减并不意味着规模报酬递减

## 特殊情况分析

### 完全替代的情况
对于线性生产函数 $y = ax_1 + bx_2$：
- 边际产品：$MP_1 = a$，$MP_2 = b$（常数，不递减）
- 技术替代率：$TRS = -\frac{a}{b}$（常数）
- 规模报酬：$y(tx_1, tx_2) = t(ax_1 + bx_2) = ty$，规模报酬不变

### 完全互补的情况
对于固定比例生产函数 $y = \min\{ax_1, bx_2\}$：
- 边际产品：在最优比例点处，边际产品为0；在非最优比例点处，边际产品为正
- 技术替代率：在最优比例点处，技术替代率未定义
- 规模报酬：$y(tx_1, tx_2) = \min\{atx_1, btx_2\} = t\min\{ax_1, bx_2\} = ty$，规模报酬不变
