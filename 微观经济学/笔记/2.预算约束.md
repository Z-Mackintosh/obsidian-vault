# 需求
消费者对于某种产品在每一个可能的价格下*愿意*（偏好）并且*能够*（预算约束）购买的数量。
# [[消费束]]
**定义**:消费者对于所决定的两种商品的购买量（点）。
**预算线**: $p_{1}x_{1}+p_{2}x_{2} = m$

> 消费束中，预算线以外的称为不可行消费束，以内的称为可行消费束。其上的点为刚好可行消费束。
# 预算集
所有可行消费束（点）的集合
## 预算线
恰好将预算花完的消费束的集合。
# 预算约束
$p_{1}x_{1}+p_{2}x_{2} \leq m$
此时斜率$-\frac{p_{1}}{p_{2}}$,表示增加一单位商品一，需要减少商品二的单位，即增加商品一消费的机会成本。
# 计价物
为了方便计算上述的斜率以更鲜明的计价，我们人为的将某一中商品的价格定位1，称之为**计价物**。此时其他商品的价格都会以计价物的基准表示。当我们把这一商品视为消费者除了商品一以外所有想消费的商品的总和，就会更符合理论。
