## 恩格尔曲线
- 收入改变但商品价格不变时商品需求量的连线（一种商品数量 $x_{1}$ 和收入水平 $m$）
- 当消费者偏好为[[相似偏好]] 时恩格尔曲线为直线
- 正常商品的恩格尔曲线斜率为正
- 劣质品/低档商品的恩格尔曲线斜率为负
### 完全替代
对于效用函数 $U(x_1, x_2) = ax_1 + bx_2$：

1. 当 $\frac{a}{b} > \frac{p_1}{p_2}$ 时：
   - 只消费 $x_1$，$x_1^* = \frac{m}{p_1}$
   - 恩格尔曲线：$m = p_1 x_1$，斜率为 $p_1$

2. 当 $\frac{a}{b} < \frac{p_1}{p_2}$ 时：
   - 只消费 $x_2$，$x_1^* = 0$
   - 恩格尔曲线：$x_1 = 0$，斜率为无穷大

3. 当 $\frac{a}{b} = \frac{p_1}{p_2}$ 时：
   - 任意消费组合满足 $p_1 x_1 + p_2 x_2 = m$
   - 恩格尔曲线：$m = p_1 x_1 + p_2 x_2$
### 完全互补
对于效用函数 $U(x_1, x_2) = \min(ax_1, bx_2)$：

- 最优消费满足：$ax_1^* = bx_2^*$
- 预算约束：$p_1 x_1^* + p_2 x_2^* = m$
- 解得：$x_1^* = \frac{bm}{bp_1 + ap_2}$，$x_2^* = \frac{am}{bp_1 + ap_2}$
- 恩格尔曲线：$m = (\frac{bp_1 + ap_2}{b}) x_1$，斜率为 $\frac{bp_1 + ap_2}{b}$
- 特殊情况：当 $a = b = 1$ 时，斜率为 $p_1 + p_2$
###  柯布道格拉斯偏好
对于效用函数 $u(x_1, x_2) = x_1^a x_2^{1-a}$：

- 最优消费：$x_1^* = \frac{am}{p_1}$，$x_2^* = \frac{(1-a)m}{p_2}$
- 恩格尔曲线：$m = \frac{p_1}{a} x_1$，斜率为 $\frac{p_1}{a}$
- 收入提供曲线：从原点出发的射线，斜率为 $\frac{p_2(1-a)}{p_1 a}$
- 支出比例固定：商品1支出占比 $a$，商品2支出占比 $1-a$
###  [[拟线性偏好]]
对于效用函数 $u(x_1, x_2) = v(x_1) + x_2$：

- 商品1需求：$x_1^* = v'^{-1}(\frac{p_1}{p_2})$（与收入无关）
- 商品2需求：$x_2^* = \frac{m - p_1 x_1^*}{p_2}$
- 恩格尔曲线：
  - 商品1：垂直直线，无收入效应
  - 商品2：$m = p_1 x_1^* + p_2 x_2$，斜率为 $p_2$
- 收入提供曲线：水平线（商品1固定，商品2随收入增加）

## 需求曲线
当收入和其他商品的价格不变时，描述价格 $p_{x}$ 和需求量 $x$ 之间的关系
  - 普通商品
$$\frac{\partial x}{\partial p_x} < 0$$
	>   大部分商品

-  吉芬商品
$$\frac{\partial x}{\partial p_x} > 0$$

  - 替代品
$$\frac{\partial x}{\partial p_y} < 0$$

-  互补品
$$\frac{\partial x}{\partial p_y} > 0$$
### 完全互补
对于效用函数 $U(x_1, x_2) = \min(ax_1, bx_2)$：

- 需求函数：$x_1^* = \frac{bm}{bp_1 + ap_2}$，$x_2^* = \frac{am}{bp_1 + ap_2}$
- 价格提供曲线：$ax_1 = bx_2$（45度线）
- 需求曲线：$x_1 = \frac{bm}{bp_1 + ap_2}$（向下倾斜）
- 交叉价格效应：$\frac{\partial x_1}{\partial p_2} < 0$（互补品）
### 完全替代
对于效用函数 $U(x_1, x_2) = ax_1 + bx_2$：

- 需求函数：
  - 当 $\frac{a}{b} > \frac{p_1}{p_2}$：$x_1^* = \frac{m}{p_1}$，$x_2^* = 0$
  - 当 $\frac{a}{b} < \frac{p_1}{p_2}$：$x_1^* = 0$，$x_2^* = \frac{m}{p_2}$
  - 当 $\frac{a}{b} = \frac{p_1}{p_2}$：任意满足预算约束
- 价格提供曲线：坐标轴
- 需求曲线：分段函数，在临界价格处跳跃
- 交叉价格效应：$\frac{\partial x_1}{\partial p_2} = 0$（独立商品）
### 柯布道格拉斯偏好
对于效用函数 $u(x_1, x_2) = x_1^a x_2^{1-a}$：

- 需求函数：$x_1^* = \frac{am}{p_1}$，$x_2^* = \frac{(1-a)m}{p_2}$
- 价格提供曲线：从原点出发的射线
- 需求曲线：$x_1 = \frac{am}{p_1}$（双曲线）
- 价格弹性：$\varepsilon = -1$（单位弹性）
- 交叉价格效应：$\frac{\partial x_1}{\partial p_2} = 0$（独立商品）

### 拟线性偏好
对于效用函数 $u(x_1, x_2) = v(x_1) + x_2$：

- 需求函数：$x_1^* = v'^{-1}(\frac{p_1}{p_2})$，$x_2^* = \frac{m - p_1 x_1^*}{p_2}$
- 价格提供曲线：垂直线（商品1固定）
- 需求曲线：$x_1 = v'^{-1}(\frac{p_1}{p_2})$（与收入无关）
- 价格弹性：取决于 $v(x_1)$ 的具体形式
- 交叉价格效应：$\frac{\partial x_1}{\partial p_2} > 0$（替代品）
