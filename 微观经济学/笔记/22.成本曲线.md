# 平均成本
总成本 = 可变成本 + 固定成本

平均成本函数度量的是每单位产量的成本，平均可变成本函数度量的是每单位产量的可变成本，平均不变成本函数度量的是每单位产量的不变成本。根据上述的等式，我们可以得到

$$
AC(y) = \frac{c(y)}{y} = \frac{c_v(y)}{y} + \frac{F}{y} = AVC(y) + AFC(y)
$$
随着产量的增加，平均不变成本减少而平均可变成本最初可能减少但最终一定会增加。总的平均成本呈U形。

# 长期成本
1. 短期成本曲线在长期成本曲线上，因为后者的固定成本得到了充分释放
2. 每一个短期成本曲线都会有一个低点，是此时的最优规模，其与长期成本曲线相切3. 综上所述，当短期成本曲线有无数条时，就会有长期成本曲线**包络**前者的情况。
# 长期边际成本
任意产量水平 $y$ 上的长期边际成本必须与生产该产量的最优工厂规模所对应的短期边际成本相等。

# 规模报酬
规模报酬不变时，长期平均成本和长期边际成本是一个常数（正常规模报酬递减，那么这两个曲线向上倾斜）。
又因为**长期供给曲线就是其长期边际成本曲线在长期平均成本曲线最低点以上的部分**，所以长期供给曲线也是常数。
