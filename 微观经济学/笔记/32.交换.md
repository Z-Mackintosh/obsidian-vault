# 契约曲线
契约曲线是埃奇沃思方框图中所有帕累托有效配置点的集合。在契约曲线上，两个消费者的无差异曲线相切，满足：
$$MRS_A = MRS_B$$

## 契约曲线的性质
1. **帕累托效率**：契约曲线上任意一点都是帕累托有效的，无法在不损害一方利益的情况下改善另一方
2. **无差异曲线相切**：在契约曲线上，两个消费者的无差异曲线相切
3. **边际替代率相等**：$MRS_A^{1,2} = MRS_B^{1,2}$
4. **从禀赋点出发**：契约曲线连接两个原点，包含所有可能的帕累托有效配置

## 契约曲线的推导
对于效用函数 $U_A(x_A^1, x_A^2)$ 和 $U_B(x_B^1, x_B^2)$，契约曲线满足：
$$\frac{\partial U_A/\partial x_A^1}{\partial U_A/\partial x_A^2} = \frac{\partial U_B/\partial x_B^1}{\partial U_B/\partial x_B^2}$$

同时满足资源约束：
$$x_A^1 + x_B^1 = \omega^1$$
$$x_A^2 + x_B^2 = \omega^2$$

其中 $\omega^1, \omega^2$ 分别是商品1和商品2的总禀赋。

## 契约曲线与竞争均衡
- 竞争均衡必然位于契约曲线上
- 不同的初始禀赋点会导致不同的竞争均衡点
- 契约曲线上的每一点都可以通过适当的财富再分配达到
# 市场交易
在整个市场交易的过程，一个消费者实际卖出或者买出的数量称之为**净需求**，或者说**净交易量**。
而而在更加真实的市场，所有消费者对于某一种商品的净需求之后就是**超额需求**。

# 均衡的代数式
我们讨论均衡，最终是为了确定一组价格。

# 瓦尔拉斯法则
$$ p_1z_1(p_1,\:p_2)+p_2z_2(p_1,\:p_2)\equiv0$$
也就是是说，总超额需求的值恒为 0。

# 相对价格
由瓦尔拉斯法则可知，对于一个含有两个价格即未知数的情况，我们只有一个方程式。推广到 $k$ 元之后任然只有 $k-1$ 个方程式，那我们如何确定价格呢。
事实上，由于价格和收入同比例防缩，预算集与需求束都不变。
所以，价格是相对的，我们可以规定一个计价价格，比方说 $1$.

# 均衡的存在性
事实上，方程有没有解都是一回事。
对数学意义的说明到实际情况上来，可以说，需要满足以下几点
1. 理性、连续、单调、凸性的偏好
2. 市场是完全竞争的（价格接受者）
3. 消费者的**初始禀赋处于消费集合内部**（Interioality）
4. 给定任意消费束 $x$，在它周围总存在一个稍有改变的消费束 $x^{'}$，让你严格更偏好。
5. 总供给足够，且无排他性或外部性
	- 自由交易，无外部性
