# 市场环境
厂商除了面临技术约束，还面临着**市场约束**，考虑到不同的**市场环境**，厂商面临着不同的需求曲线。
这章我们讨论完全竞争。
# 完全竞争
- 许多小厂商而且生产同质的产品
- 厂商为价格的接受者

## 需求曲线
- 市场需求曲线
    度量市场价格和销售总量
    取决与消费者行为
- 厂商面临的需求曲线
    市场价格和特定厂商的产量
    还取决于其他厂商……

# 供给决策
- 边际成本 = 价格
> 但是这只是厂商实现利润最大化的必要条件，考虑到吉芬商品等因素，其没有实现充分性

- 如果边际成本等于价格的点位于平均可变成本曲线的下发，厂商的最佳产量为0

# 反供给函数
- 价格作为产量的函数
- 市场价格描述厂商的边际成本

# 长期不变的平均成本
规模报酬不变的时候，企业的长期平均成本 = 长期边际成本
