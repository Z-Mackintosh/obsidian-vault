# 题目
一个双寡头卖方垄断面临的反需求曲线为 $p=160-2q$，市场中的两个企业生产的成本固定为$10/单位。在古诺均衡中，每个寡头将要卖出多少产品？

# 解答

**已知：**
- 反需求函数：$p = 160 - 2q$，其中 $q = q_1 + q_2$
- 边际成本：$MC = 10$（固定成本为$10/单位）
- 两个企业：企业1和企业2

**求解步骤：**

1. **企业1的利润函数：**
   $\pi_1 = p \cdot q_1 - 10q_1 = (160 - 2(q_1 + q_2))q_1 - 10q_1$

2. **企业1的一阶条件（利润最大化）：**
   $\frac{\partial \pi_1}{\partial q_1} = 160 - 2(q_1 + q_2) - 2q_1 - 10 = 0$
   $150 - 4q_1 - 2q_2 = 0$
   $4q_1 = 150 - 2q_2$
   $q_1 = \frac{150 - 2q_2}{4}$ （企业1的反应函数）

3. **企业2的利润函数（对称）：**
   $\pi_2 = (160 - 2(q_1 + q_2))q_2 - 10q_2$

4. **企业2的一阶条件：**
   $\frac{\partial \pi_2}{\partial q_2} = 160 - 2(q_1 + q_2) - 2q_2 - 10 = 0$
   $150 - 2q_1 - 4q_2 = 0$
   $4q_2 = 150 - 2q_1$
   $q_2 = \frac{150 - 2q_1}{4}$ （企业2的反应函数）

5. **求解古诺均衡：**
   将企业2的反应函数代入企业1的反应函数：
   $q_1 = \frac{150 - 2\left(\frac{150 - 2q_1}{4}\right)}{4}$
   $q_1 = \frac{150 - \frac{150 - 2q_1}{2}}{4}$
   $q_1 = \frac{150 - 75 + q_1}{4}$
   $q_1 = \frac{75 + q_1}{4}$
   $4q_1 = 75 + q_1$
   $3q_1 = 75$
   $q_1 = 25$

6. **由对称性得：**
   $q_2 = 25$

**答案：**
在古诺均衡中，每个寡头将卖出 **25单位** 产品。

**验证：**
- 总产量：$q = q_1 + q_2 = 50$
- 市场价格：$p = 160 - 2 \times 50 = 60$
- 每个企业的利润：$\pi = (60 - 10) \times 25 = 1250$
