
# 题目
瑞驰是美国中西部一个小城市具有控制地位的二手车经销商。支付过$50 000 的管理费用后，瑞驰的成本为$500/辆。城里还有 5 家其他的小经销商，但由于他们不能像瑞驰那样经由同样的折扣渠道获得汽车，他们的成本函数均为 $C=5000+700Q+5Q^2$。二手车的需求函数为 $Q=200-\frac{P1}{10}$。假设瑞驰知道其竞争对手的成本，瑞驰将要给每辆车定价多少？

# 解答

## 1. 市场结构分析
这是一个**主导厂商模型**（dominant firm model）：
- 瑞驰是主导厂商（dominant firm）
- 其他5家小经销商是竞争性边缘厂商（competitive fringe）

## 2. 边缘厂商供给曲线
对于每个边缘厂商，成本函数为：
$$C = 5000 + 700Q + 5Q^2$$
边际成本为：
$$MC = \frac{dC}{dQ} = 700 + 10Q$$

在完全竞争市场中，边缘厂商接受市场价格 $P$ 并生产到 $P = MC$：
$$P = 700 + 10Q_f$$
其中 $Q_f$ 是单个边缘厂商的产量。

解出单个边缘厂商的供给函数：
$$Q_f = \frac{P - 700}{10}$$

由于有5个相同的边缘厂商，总边缘供给为：
$$Q_f^{total} = 5 \times \frac{P - 700}{10} = \frac{P - 700}{2}$$

## 3. 市场需求函数
市场需求为：
$$Q = 200 - \frac{P}{10}$$
或逆需求函数：
$$P = 2000 - 10Q$$

## 4. 主导厂商的剩余需求
主导厂商面临剩余需求（residual demand）：
$$Q_d = Q_{market} - Q_f^{total} = (200 - \frac{P}{10}) - \frac{P - 700}{2}$$

简化：
$$Q_d = 200 - \frac{P}{10} - \frac{P}{2} + 350 = 550 - \frac{6P}{10} = 550 - 0.6P$$

逆剩余需求函数：
$$P = \frac{550 - Q_d}{0.6} = 916.67 - 1.667Q_d$$

## 5. 瑞驰的利润最大化
瑞驰的成本：
- 固定成本：$50,000$
- 边际成本：$500$（常数）

利润函数：
$$\pi = P \times Q_d - 500Q_d - 50000$$
代入逆剩余需求：
$$\pi = (916.67 - 1.667Q_d) \times Q_d - 500Q_d - 50000$$
$$\pi = 916.67Q_d - 1.667Q_d^2 - 500Q_d - 50000$$
$$\pi = 416.67Q_d - 1.667Q_d^2 - 50000$$

一阶条件：
$$\frac{d\pi}{dQ_d} = 416.67 - 3.334Q_d = 0$$
解得：
$$Q_d = \frac{416.67}{3.334} \approx 125$$

## 6. 最优价格
代入逆剩余需求函数：
$$P = 916.67 - 1.667 \times 125 = 916.67 - 208.375 = 708.295 \approx 708.30$$

## 7. 验证
边缘厂商总产量：
$$Q_f^{total} = \frac{P - 700}{2} = \frac{708.30 - 700}{2} = \frac{8.30}{2} = 4.15$$

总市场产量：
$$Q_{market} = 200 - \frac{P}{10} = 200 - 70.83 = 129.17$$

检查：$Q_d + Q_f^{total} = 125 + 4.15 = 129.15 \approx 129.17$（四舍五入误差）

## 8. 答案
瑞驰的最优定价为 **$708.30**。

此时：
- 瑞驰产量：125辆
- 边缘厂商总产量：4.15辆
- 市场总产量：129.17辆
- 瑞驰利润：$\pi = 416.67 \times 125 - 1.667 \times 125^2 - 50000 = 52083.75 - 26046.875 - 50000 = -23963.125$

注意：瑞驰在此价格下实际上有亏损，这是因为固定成本$50,000$很高。但在短期，只要价格高于平均可变成本（$500$），瑞驰仍会继续生产。
