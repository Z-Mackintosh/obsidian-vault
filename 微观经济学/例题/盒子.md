# 题目
考虑一个包含两种商品和两个消费者的纯交换经济。消费者1的初始禀赋为 $(\omega_{11}, \omega_{12}) = (7, 3)$，效用函数为 $u_1(x_{11}, x_{12}) = x_{11}^{2} x_{12}$；消费者2的初始禀赋为 $(\omega_{21}, \omega_{22}) = (2/3, 4)$，效用函数为 $u_2(x_{21}, x_{22}) = 4\sqrt{x_{21}} + 2x_{22}$。画出帕累托改进区间和交易契约曲线。求竞争均衡的商品价格组合和均衡配置（把商品1的价格标准化为1）。

# 解答

## 1. 初始禀赋和总资源

消费者1: $(\omega_{11}, \omega_{12}) = (7, 3)$  
消费者2: $(\omega_{21}, \omega_{22}) = (2/3, 4)$  

总资源:  
商品1: $7 + 2/3 = 23/3$  
商品2: $3 + 4 = 7$  

## 2. 帕累托改进区间

帕累托改进是指在不损害任何消费者福利的前提下，至少有一个消费者的福利得到改善。

对于消费者1，初始效用: $u_1(7, 3) = 7^2 \times 3 = 147$  
对于消费者2，初始效用: $u_2(2/3, 4) = 4\sqrt{2/3} + 2\times4 = 4\sqrt{2/3} + 8 \approx 11.27$  

帕累托改进区间由以下条件定义：
- $u_1(x_{11}, x_{12}) \geq 147$
- $u_2(x_{21}, x_{22}) \geq 11.27$
- $x_{11} + x_{21} = 23/3$
- $x_{12} + x_{22} = 7$

## 3. 交易契约曲线

契约曲线是帕累托最优配置的集合，满足边际替代率相等：

消费者1的边际替代率：
$$MRS_1 = \frac{\partial u_1/\partial x_{11}}{\partial u_1/\partial x_{12}} = \frac{2x_{11}x_{12}}{x_{11}^2} = \frac{2x_{12}}{x_{11}}$$

消费者2的边际替代率：
$$MRS_2 = \frac{\partial u_2/\partial x_{21}}{\partial u_2/\partial x_{22}} = \frac{2/\sqrt{x_{21}}}{2} = \frac{1}{\sqrt{x_{21}}}$$

契约曲线条件：$MRS_1 = MRS_2$
$$\frac{2x_{12}}{x_{11}} = \frac{1}{\sqrt{x_{21}}}$$

代入资源约束：$x_{21} = 23/3 - x_{11}$, $x_{22} = 7 - x_{12}$

得到契约曲线方程：
$$\frac{2x_{12}}{x_{11}} = \frac{1}{\sqrt{23/3 - x_{11}}}$$

## 4. 竞争均衡

设商品1价格 $p_1 = 1$，商品2价格 $p_2 = p$

### 消费者1的最优选择
预算约束：$x_{11} + px_{12} = 7 + 3p$

效用最大化：$\max u_1 = x_{11}^2 x_{12}$
一阶条件：$MRS_1 = \frac{p_1}{p_2} = \frac{1}{p}$
$$\frac{2x_{12}}{x_{11}} = \frac{1}{p} \Rightarrow x_{12} = \frac{x_{11}}{2p}$$

代入预算约束：
$$x_{11} + p\cdot\frac{x_{11}}{2p} = 7 + 3p$$
$$x_{11} + \frac{x_{11}}{2} = 7 + 3p$$
$$\frac{3}{2}x_{11} = 7 + 3p$$
$$x_{11} = \frac{2}{3}(7 + 3p)$$
$$x_{12} = \frac{1}{2p} \cdot \frac{2}{3}(7 + 3p) = \frac{7 + 3p}{3p}$$

### 消费者2的最优选择
预算约束：$x_{21} + px_{22} = 2/3 + 4p$

效用最大化：$\max u_2 = 4\sqrt{x_{21}} + 2x_{22}$

内点解条件：$MRS_2 = \frac{p_1}{p_2} = \frac{1}{p}$
$$\frac{1}{\sqrt{x_{21}}} = \frac{1}{p} \Rightarrow \sqrt{x_{21}} = p \Rightarrow x_{21} = p^2$$

代入预算约束：
$$p^2 + px_{22} = 2/3 + 4p$$
$$x_{22} = \frac{2/3 + 4p - p^2}{p}$$

### 市场出清条件
商品1市场：$x_{11} + x_{21} = 23/3$
$$\frac{2}{3}(7 + 3p) + p^2 = 23/3$$
$$\frac{14 + 6p}{3} + p^2 = 23/3$$
$$14 + 6p + 3p^2 = 23$$
$$3p^2 + 6p - 9 = 0$$
$$p^2 + 2p - 3 = 0$$
$$(p + 3)(p - 1) = 0$$

由于价格为正，$p = 1$

### 均衡配置
当 $p = 1$ 时：

消费者1：
$$x_{11} = \frac{2}{3}(7 + 3\times1) = \frac{2}{3}\times10 = \frac{20}{3}$$
$$x_{12} = \frac{7 + 3\times1}{3\times1} = \frac{10}{3}$$

消费者2：
$$x_{21} = 1^2 = 1$$
$$x_{22} = \frac{2/3 + 4\times1 - 1^2}{1} = \frac{2}{3} + 4 - 1 = \frac{11}{3}$$

验证：
商品1总量：$20/3 + 1 = 23/3$ ✓
商品2总量：$10/3 + 11/3 = 21/3 = 7$ ✓

## 5. 最终答案

- **竞争均衡价格**：$(p_1, p_2) = (1, 1)$
- **均衡配置**：
  - 消费者1：$(x_{11}, x_{12}) = (20/3, 10/3)$
  - 消费者2：$(x_{21}, x_{22}) = (1, 11/3)$

- **帕累托改进区间**：满足 $u_1 \geq 147$ 且 $u_2 \geq 11.27$ 的所有可行配置
- **契约曲线**：$\frac{2x_{12}}{x_{11}} = \frac{1}{\sqrt{23/3 - x_{11}}}$，其中 $0 < x_{11} < 23/3$，$0 < x_{12} < 7$
