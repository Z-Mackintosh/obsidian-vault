## 1. 判断题

**a. 在相似偏好情况下，收入不变，所有商品的价格加倍，那么对所有商品的需求减半**

**正确**。相似偏好意味着消费比例固定，当所有价格同比例变化时，实际购买力不变，需求数量与价格成反比。

**b. 在其他变量固定的情况下，收入增加，吉芬商品的需求减少**

**正确**。吉芬商品是低档商品，收入增加时需求减少，且其收入效应超过替代效应，导致价格下降时需求反而减少。

**c. 小张喜欢音乐，且每次都会买够2盘磁带和1张唱片，唱片价格20元/张，磁带10元/盘。当唱片价格下降到10元/张时，他购买的唱片数量的变化全部来自于替代效应。**

**错误**。这是完全互补偏好，无差异曲线呈L形。价格下降时只有收入效应，没有替代效应，因为消费者总是按固定比例消费。

**d. 若消费者在任何收入和价格条件下，总是用30%的支出用于住房，30%用于饮食，10%用于购买服装，10%用于交通，20%用于休闲，这种行为说明这个消费者满足柯布道格拉斯偏好。**

**正确**。柯布道格拉斯偏好的特点是支出份额固定，与笔记中"柯布-道格拉斯偏好是相似偏好"一致。

## 2. 某消费者的效用函数为 $u(x,y) = \sqrt{x} + \sqrt{y}$

假设他的收入为100元，商品y的价格为1元/单位。

### （1）Slutsky替代效应和收入效应

价格变化：商品x的价格从4元/单位下降到1元/单位

**初始状态**：$p_x=4, p_y=1, m=100$

**最优消费条件**：$\frac{MU_x}{p_x} = \frac{MU_y}{p_y}$ ⇒ $\frac{1/(2\sqrt{x})}{p_x} = \frac{1/(2\sqrt{y})}{p_y}$ ⇒ $\frac{\sqrt{y}}{\sqrt{x}} = \frac{p_x}{p_y}$

初始最优：$\frac{\sqrt{y}}{\sqrt{x}} = \frac{4}{1}$ ⇒ $y=16x$
预算约束：$4x + y = 100$ ⇒ $4x + 16x = 100$ ⇒ $x=5, y=80$

**最终状态**：$p_x=1, p_y=1, m=100$

最终最优：$\frac{\sqrt{y}}{\sqrt{x}} = \frac{1}{1}$ ⇒ $x=y$
预算约束：$x + y = 100$ ⇒ $x=50, y=50$

**Slutsky替代效应**：
保持购买力不变，$m' = m + x\Delta p_x = 100 + 5(-3) = 85$
在$p_x=1, p_y=1, m'=85$时：$x=y$ ⇒ $2x=85$ ⇒ $x=42.5$
替代效应：$42.5 - 5 = 37.5$

**收入效应**：$50 - 42.5 = 7.5$

### （2）Hicks替代效应和收入效应

Hicks替代效应保持效用不变：
初始效用：$U_0 = \sqrt{5} + \sqrt{80} = \sqrt{5} + 4\sqrt{5} = 5\sqrt{5}$
在$p_x=1, p_y=1$下最小化支出：$\sqrt{x} + \sqrt{y} = U_0, x=y$ ⇒ $2\sqrt{x} = 5\sqrt{5}$ ⇒ $\sqrt{x} = \frac{5\sqrt{5}}{2}$ ⇒ $x = \frac{125}{4} = 31.25$
Hicks替代效应：$31.25 - 5 = 26.25$
收入效应：$50 - 31.25 = 18.75$

## 3. 某消费者效用函数为$u(x,y) = \sqrt{x} + y$

**收入提供曲线**：
$\frac{MU_x}{p_x} = \frac{MU_y}{p_y}$ ⇒ $\frac{1/(2\sqrt{x})}{p_x} = \frac{1}{p_y}$ ⇒ $\sqrt{x} = \frac{p_y}{2p_x}$ ⇒ $x = (\frac{p_y}{2p_x})^2$

x的需求与收入无关（零收入效应），y的需求随收入增加而增加。

**恩格尔曲线**：
- x的恩格尔曲线是水平线
- y的恩格尔曲线是斜率为1的直线

**价格提供曲线**：
x的价格提供曲线是垂直的，因为x的需求只取决于价格比。

**需求曲线**：
- x的需求曲线：$x = (\frac{p_y}{2p_x})^2$
- y的需求曲线：$y = m - p_x(\frac{p_y}{2p_x})^2$
