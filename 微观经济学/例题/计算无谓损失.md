# 例题  
Billy-Bob 的流动加油站的每日汽油需求量可由 $Q = 776 - 200 p$ 表示，$Q$ 为卖出的汽油加仑数，$p$ 为汽油价格，以美元衡量。Billy-Bob 流动加油站的供给为 $Q = 890 + 1500 p$。假设州政府决定对售出的每加仑汽油征收 20 美分的税，则由税收造成的无谓损失为多少？

# 解答  
## 方法分析  
本题要求计算税收造成的无谓损失（Deadweight Loss）。无谓损失是由于税收导致市场交易量减少，从而损失的部分社会总剩余。计算无谓损失通常需要先求出征税前的市场均衡价格和数量，再分析征税后消费者支付价格与生产者获得价格的变化，以及由此导致的数量变化。无谓损失可以近似为三角形面积：$\frac{1}{2} \times \text{单位税额} \times \text{减少的交易量}$。

## 解题步骤  
### Step 1：求解征税前的市场均衡  
需求函数：$Q_d = 776 - 200p$  
供给函数：$Q_s = 890 + 1500p$  
令 $Q_d = Q_s$ 求均衡：  
$$776 - 200p = 890 + 1500p$$  
$$776 - 890 = 1500p + 200p$$  
$$-114 = 1700p$$  
$$p^* = -\frac{114}{1700} \approx -0.0671$$  
这里出现负价格，表明原函数设定可能将价格 $p$ 视为实际交易价格（非负），但数学求解得到负值，说明函数形式或常数项可能基于特定背景（如包含补贴或特定区间）。但根据经济学意义，价格应为非负，因此我们应检查函数。实际上，需求函数斜率为负，供给函数斜率为正，常数项差异导致负均衡价格，这可能是因为题目中的函数已经隐含了某种基准（如含税或补贴）。但题目要求计算税收效应，我们可以直接利用函数斜率进行分析，因为无谓损失计算主要依赖于斜率。

### Step 2：分析税收影响与数量变化  
设征税前均衡价格为 $p_0$，均衡数量为 $Q_0$。征税后，消费者支付价格 $p_d$，生产者获得价格 $p_s$，且 $p_d = p_s + t$，其中 $t = 0.2$ 美元。  
需求函数：$Q = 776 - 200 p_d$  
供给函数：$Q = 890 + 1500 p_s$  
在征税后新均衡下，需求量等于供给量，且 $p_d = p_s + 0.2$。  
将 $p_d = p_s + 0.2$ 代入需求函数：  
$$Q = 776 - 200(p_s + 0.2) = 776 - 200p_s - 40 = 736 - 200p_s$$  
令其等于供给函数：  
$$736 - 200p_s = 890 + 1500p_s$$  
$$736 - 890 = 1500p_s + 200p_s$$  
$$-154 = 1700p_s$$  
$$p_s = -\frac{154}{1700} \approx -0.0906$$  
则 $p_d = p_s + 0.2 \approx 0.1094$。  
征税后交易量：$Q_1 = 890 + 1500 \times (-0.0906) \approx 890 - 135.9 = 754.1$，或从需求函数得 $Q_1 = 736 - 200 \times (-0.0906) \approx 736 + 18.12 = 754.12$。  
为求减少的交易量，需要知道征税前均衡量。从 Step 1，$p^* \approx -0.0671$，则 $Q_0 = 776 - 200 \times (-0.0671) = 776 + 13.42 = 789.42$。  
减少的交易量 $\Delta Q = Q_0 - Q_1 \approx 789.42 - 754.12 = 35.3$。  

但原解答采用了一种简化方法：直接利用斜率关系。需求曲线斜率绝对值为 $\frac{1}{200}$（价格变化对数量的影响），供给曲线斜率为 $\frac{1}{1500}$。设税收导致交易量减少 $x$，则消费者价格上升部分为 $\frac{x}{200}$，生产者价格下降部分为 $\frac{x}{1500}$，两者之和应等于单位税额 $t=0.2$，即：  
$$\frac{x}{200} + \frac{x}{1500} = 0.2$$  
此方程基于线性函数斜率与价格变化关系。

### Step 3：计算减少的交易量  
解方程：  
$$\frac{x}{200} + \frac{x}{1500} = 0.2$$  
通分：$\frac{15x}{3000} + \frac{2x}{3000} = 0.2$  
$$\frac{17x}{3000} = 0.2$$  
$$x = 0.2 \times \frac{3000}{17} = \frac{600}{17} \approx 35.294$$

### Step 4：计算无谓损失  
无谓损失（DWL）为三角形面积：  
$$\text{DWL} = \frac{1}{2} \times t \times x = \frac{1}{2} \times 0.2 \times \frac{600}{17} = \frac{60}{17} \approx 3.5294$$  
因此，无谓损失约为 3.53 美元。

最终答案：$\boxed{3.53}$

# 核心技巧  
## 税收无谓损失计算  
- 对于线性供求曲线，无谓损失可直接用公式 $\text{DWL} = \frac{1}{2} \times t \times \Delta Q$ 计算，其中 $\Delta Q$ 是税收导致的数量减少。  
- $\Delta Q$ 可通过税收分摊与斜率关系求得：设需求曲线斜率为 $k_d$（绝对值），供给曲线斜率为 $k_s$，则 $\Delta Q = t \times \frac{1}{\frac{1}{k_d} + \frac{1}{k_s}} = t \times \frac{k_d k_s}{k_d + k_s}$。本题中 $k_d = 200$, $k_s = 1500$，代入即得。

# 经验总结  
1. **无谓损失的本质**：税收扭曲市场价格信号，导致交易量减少，损失了部分互惠交易的社会福利，其大小与供求弹性（斜率）相关。  
2. **线性模型简化计算**：在线性供求模型中，无谓损失三角形面积计算只需单位税额和数量变化，数量变化可通过斜率与税收分摊快速求出，避免求解具体价格。  
3. **注意函数定义域**：本题中数学求解出现负价格，表明函数可能定义在特定区间或包含隐含条件，但无谓损失计算仅依赖斜率时可不深究原均衡，直接利用斜率关系更高效。
