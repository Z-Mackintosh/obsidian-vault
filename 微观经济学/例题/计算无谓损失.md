# 题目
Billy-Bob 的流动加油站的每日汽油需求量可由 $Q = 776 - 200 p$ 表示，$Q$ 为卖出的汽油加仑数，$p$ 为汽油价格，以美元衡量。Billy-Bob 流动加油站的供给为 $Q = 890 + 1500 p$。假设州政府决定对售出的每加仑汽油征收 20 美分的税，则由税收造成的无谓损失为多少？

# 解答 
假设税收使得均衡销售量减少 $x$，则：
由消费者承担的税负为 $\frac{x}{200}$，生产者承担的税负为 $\frac{x}{1500}$。
所以，由题意可得 $\frac{x}{1500} + \frac{x}{200} = 0.2 \Rightarrow x = \frac{600}{17}$
因此，无谓损失为 $\frac{1}{2} \times 0.2 \times \frac{600}{17} \approx 3.53$
