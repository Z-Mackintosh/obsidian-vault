# 二维随机变量
# **定义**：
联合分布函数：设 $(X, Y)$是二维随机变量，对于任意实数$x, y$，二元函数
$$
F(x, y) = P\{X \leq x, Y \leq y\}
$$
称为二维随机变量 $(X, Y)$的分布函数，或称为随机变量$X$和$Y$的联合分布函数，它表示随机事件$\{X \leq x\}$与$\{Y \leq y\}$ 同时发生的概率。

# **性质**：

1) 非负性：任意实数 $x, y$，$0 \leq F (x, y) \leq 1$

2) 规范性：对于任意固定的 $y$，$F (-\infty, y) = \lim_{x \to -\infty} F (x, y) = 0$，对于任意固定的 $x$，$F (x, -\infty) = \lim_{y \to -\infty} F (x, y) = 0$，$F (-\infty, -\infty) = 0$，$F (+\infty, +\infty) = 1$

3) 单调不减性：$F (x, y)$关于变量$x$和$y$单调不减，即对于任意固定的$y$，当 $x_2 > x_1$ 时，$F (x_2, y) \geq F (x_1, y)$；对于任意固定的 $x$，当 $y_2 > y_1$ 时，$F (x, y_2) \geq F (x, y_1)$。

4) 右连续性：$F (x, y)$关于变量$x$右连续，关于$y$ 也右连续，即
$$F(x + 0, y) = F(x, y)  ,F(x, y + 0) = F(x, y)$$
5) 对于任意 $x_1 < x_2$，$y_1 < y_2$，下述不等式成立：
 $$P\{x_1 < X \leq x_2, \quad y_1 < Y \leq y_2\} = F(x_2, y_2) - F(x_2, y_1) - F(x_1, y_2) + F(x_1, y_1) \geq 0$$
 即$F (x_2, y_2) + F (x_1,  y_1) \geq F (x_2, y_1) + F (x_1, y_2)$
# 边缘分布函数

定义：记 $X$ 和 $Y$ 的分布函数为 $F_X (x)$ 和 $F_Y (y)$，分别称它们为二维随机变量 $(X, Y)$ 关于 $X$ 和关于 $Y$ 的边缘分布函数。其中

$$F_X(x) = P\{X \leq x\} = P\{X \leq x, \quad Y < +\infty\} = F(x, +\infty)$$


即 $F_X (x) = F (x, +\infty)$，同理 $F_Y (y) = F (+\infty, y)$。

注：边缘分布实际上是高维随机变量的某个 (某些) 低维分量的分布。
# 独立性
## 定义：
设 $X, Y$ 是两个随机变量，如果对于任意的实数 $x, y$ 有
$$P\{X \leq x, Y \leq y\} = P\{X \leq x\} \cdot P\{Y \leq y\}$$

即 $F (x, y) = F_X (x) \cdot F_Y (y)$，则称随机变量 $X$ 和 $Y$ 相互独立。

## 推广：
设 $X_1, X_2, \cdots, X_n$ 为 $n$ 个随机变量，如果对于任意实数 $x_1, x_2, \cdots, x_n$，有
$$F(x_1, x_2, \cdots, x_n) = F_{X_1}(x_1) F_{X_2}(x_2) \cdots F_{X_n}(x_n)$$
则称 $X_1, X_2, \cdots, X_n$ 相互独立。
## 性质 ：
设 $X_1, X_2, \cdots, X_n$ 相互独立，且 $g_1 (x), g_2 (x), \cdots, g_n (x)$ 均为连续函数，则 $g_1 (X_1), g_2 (X_2), \cdots, g_n (X_n)$ 也相互独立。
