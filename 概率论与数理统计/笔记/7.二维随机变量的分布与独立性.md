# 联合分布律
……所有可能的取值……称为二维随机变量 $(X,Y)$ 的分布律或者 $X$ 和 $Y$ 的联合分布律

# 联合分布函数
离散型随机变量 $(X, Y)$ 的联合分布函数
$$
F(x, y) = P\{X \leq x, Y \leq y\} = \sum_{x_i \leq x} \sum_{y_j \leq y} p_{ij}
$$
即对一切满足不等式 $x_i \leq x$, $y_j \leq y$ 的 $p_{ij}$ 求和. 
# 边缘分布律
## 定义
设 $(X, Y)$ 是二维离散型随机变量，设其概率分布为
$$ P\{X = x_i, Y = y_j\} = p_{ij}, \quad i, j = 1, 2, \ldots $$
则 $X$ 的边缘分布律为：
$$ P\{X = x_i\} = P\{X = x_i, \quad Y < +\infty\} = \sum_{j=1}^{\infty} P\{X = x_i, Y = y_j\} = \sum_{j=1}^{\infty} p_{ij} = p_i, \quad i = 1, 2, \ldots $$
$X$ 的边缘分布函数为：
$$ F_X(x) = P\{X \leq x\} = \sum_{x_i \leq x} P\{X \leq x_i\} = \sum_{x_i \leq x} p_i $$
同理有 $Y$ 的边缘分布律
# 条件分布律
## 定义
设 $(X, Y)$ 是二维离散型随机变量，对于固定的 $j$，若 $P\{Y = y_j\} > 0$，则称
$$ P\{X = x_i \mid Y = y_j\} = \frac{P\{X = x_i, Y = y_j\}}{P\{Y = y_j\}} = \frac{p_{ij}}{p_j}, i = 1, 2, \ldots $$
为在 $Y = y_j$ 条件下随机变量 $X$ 的条件分布律。
同样，对于固定的 $i$，若 $P\{X = x_i\} > 0$，则称
$$ P\{Y = y_j \mid X = x_i\} = \frac{P\{X = x_i, Y = y_j\}}{P\{X = x_i\}} = \frac{p_{ij}}{p_i}, j = 1, 2, \ldots $$
为在 $X = x_i$ 条件下随机变量 $Y$ 的条件分布律。
# 独立性
对于二维离散型随机变量 $(X, Y)$，则 $X$ 与 $Y$ 相互独立的充要条件是：
$$ P\{X = x_i, Y = y_j\} = P\{X = x_i\} \cdot P\{Y = y_j\}, i, j = 1, 2, \ldots $$
即 $p_{ij} = p_i \cdot p_j, (i, j = 1, 2, \ldots)$ 
>  **技巧**：
> 	 1. $p_{ij}=0,变量不独立$
> 	 2. 如果独立，那么联合分布律的矩阵的秩为 1.
