# 概念和性质
## 概念
- 对于随机变量 $X$的分布函数$F (x)$，如果存在非负可积函数 $f (x)$，使对任意实数 $x$ 有

$$F(x) = \int_{-\infty}^{x} f(t) dt$$

则 $X$称为**连续型随机变量**，其中$f (x)$称为$X$ 的概率密度函数，简称**概率密度**。

>  注
1) 连续型 $X$的概率密度$f (x)$包含了$X$ 所有取值概率的信息。

2) 由定义知，改变概率密度 $f (x)$在有限点的函数值不影响分布函数$F (x)$ 的取值，因此，并不在乎改变概率密度在有限点的值。

3) 若不计高阶无穷小，有

$$P\{x < X \leq x + \Delta x\} = F(x + \Delta x) - F(x) \approx f(x) \Delta x$$
即 $f (x)$的大小能反映出$X$在$x$ 邻域内取值概率的大小（“连续化的分布律”）。
4) 当我们提到一个随机变量 $X$的概率分布时，指的是它的分布函数，或者，当$X$是连续型随机变量时，指的是它的概率密度，当$X$ 是离散型随机变量时，指的是它的分布律。

5) $f (x)$ 不一定连续（只需可积）。

6. 连续型随机变量 $X$的分布函数$F (x)$ 的性质：

- 连续型随机变量 $X$的分布函数$F (x)$是$x$的连续函数，即对任意$a \in R$，有

$$P\{X = a\} = 0$$


$$P\{a < X \leq b\} = P\{a \leq X < b\} = P\{a \leq X \leq b\} = P\{a < X < b\}$$


注：分布函数 $F (x)$不连续的随机变量$X$ 一定不是连续型随机变量。
## 性质

1. 非负性：$f (x) \geq 0$。

2. 规范性：$\int_{-\infty}^{+\infty} f (x) dx = 1$。

3. 对于任意实数 $a, b$，$a \leq b$，有 $P\{a < X \leq b\} = F (b) - F (a) = \int_{a}^{b} f (x) dx$。

4. 若 $f (x)$在点$x$连续，则有$F' (x) = f (x)$。

# 分类
## 均匀分布
若连续型随机变量 $X$ 的概率密度为
$$ f(x) = \begin{cases}
\frac{1}{b-a}, & a < x < b, \\
0, & \text{其他},
\end{cases} $$

则称 $X$ 在区间 $(a, b)$ 上服从均匀分布，记为 $X \sim U (a, b)$.

- **要求**：
	- 了解均匀分布形式
	- 求解均匀分布型的分布函数
	- 通过长度比求解概率

## 指数分布
### 定义；形式
设连续型随机变量 $X$ 的概率密度为
$$
f(x)=\left\{
\begin{array}{ll}
\lambda e^{-\lambda x}, & x>0, \\
0, & x \leq 0,
\end{array}
\right.
$$
其中 $\lambda>0$为常数，则称$X$服从参数为$\lambda$的指数分布，记为$X \sim E(\lambda)$.

$X$ 的分布函数为
$$
F(x)=\left\{
\begin{array}{ll}
1-e^{-\lambda x}, & x \geq 0, \\
0, & x < 0.
\end{array}
\right.
$$
### 无记忆性
$$ P\begin{Bmatrix}X>s+t\mid X>s\end{Bmatrix}=P\begin{Bmatrix}X>t\end{Bmatrix}$$
## 正态分布
### 初认识
**基本定义**
设连续型随机变量 $X$ 的概率密度为
$$f(x) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}}, \quad -\infty < x < +\infty$$

其中 $\mu, \sigma (\sigma > 0)$ 为常数，则称 $X$ 服从参数为 $\mu, \sigma$ 的正态（Normal）分布或高斯（Gauss）分布，记为 $X \sim N (\mu, \sigma^2)$.

**基本图形性质**
1) 曲线关于 $x = \mu$ 对称，且当 $x = \mu$ 时，$f (x)$ 取最大值 $f (x) = \frac{1}{\sqrt{2\pi}\sigma}$；
2) 曲线以 $Ox$ 轴为水平渐近线。

**基本结论**
1) $P\{X > \mu\} = P\{X < \mu\} = \frac{1}{2}$
2) $Y = aX + b \sim N (a\mu + b, a^2\sigma^2)$，$a \neq 0$，即正态随机变量的一次函数仍服从正态分布。
3) $3\sigma$ 原理（了解）：
$$P\{\mu - \sigma \leq X \leq \mu + \sigma\} = 0.6827$$
$$P\{\mu - 2\sigma \leq X \leq \mu + 2\sigma\} = 0.9545$$
$$P\{\mu - 3\sigma \leq X \leq \mu + 3\sigma\} = 0.9973$$
### 标准正态分布
**定义**
当 $\mu = 0, \sigma = 1$时，称$X$服从标准正态分布，记为$X \sim N(0, 1)$，其概率密度为 $\varphi(x) = \frac{1}{\sqrt{2\pi}} e^{-\frac{x^2}{2}}, -\infty < x < +\infty$，分布函数用 $\Phi(x)$ 表示。
**性质**
- $\Phi(-a) = 1 - \Phi(a)$，$\forall a \in R$；
- $\Phi(0) = \frac{1}{2}$；
- $P\{|X| \leq a\} = 2\Phi(a) - 1$，其中 $a > 0$。
**标准化**
- 若 $X \sim N (\mu, \sigma^2)$，则 $Z = \frac{X - \mu}{\sigma} \sim N (0, 1)$。
- 性质
	- $F(x)=P\{X\leq x\}=P\left\{\frac{X-\mu}{\sigma}\leq\frac{x-\mu}{\sigma}\right\}=\Phi\left(\frac{x-\mu}{\sigma}\right)$
	- $\bullet\quad P\{a<X\leq b\}=P\left\{\frac{a-\mu}{\sigma}<\frac{X-\mu}{\sigma}\leq\frac{b-\mu}{\sigma}\right\}=\Phi\left(\frac{b-\mu}{\sigma}\right)-\Phi\left(\frac{a-\mu}{\sigma}\right)$
