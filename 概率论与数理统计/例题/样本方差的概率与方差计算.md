# 题目
设在总体 $N(\mu, \sigma^2)$ 中抽得一容量为 16 的样本，这里 $\mu, \sigma^2$ 均未知。
1. 求 $P\{S^2/\sigma^2 \leq 2.041\}$，其中 $S^2$ 为样本方差。
2. 求 $D(S^2)$

# 解答

**已知**：总体 $N(\mu, \sigma^2)$，样本容量 $n = 16$

根据 [[笔记/数字特征与参数估计/4. 样本以及抽样分布.md|4. 样本以及抽样分布]] 中的定理：

$$\frac{(n-1)S^2}{\sigma^2} = \frac{\sum_{i=1}^{n} (X_i - \overline{X})^2}{\sigma^2} \sim \chi^2(n-1)$$

## 第1问：求 $P\{S^2/\sigma^2 \leq 2.041\}$

设 $\chi^2 = \frac{(n-1)S^2}{\sigma^2} \sim \chi^2(15)$

$$P\left\{\frac{S^2}{\sigma^2} \leq 2.041\right\} = P\left\{\frac{15S^2}{\sigma^2} \leq 30.615\right\} = P\{\chi^2 \leq 30.615\}$$

根据 [[笔记/数字特征与参数估计/4. 样本以及抽样分布.md|$\chi^2$ 分布的上分位数]] 的定义 $P\{\chi^2 > \chi^2_\alpha(n-1)\} = \alpha$，查表得：
$\chi^2_{0.01}(15) = 30.578 \approx 30.615$

因此：
$$P\{\chi^2 \leq 30.615\} = 1 - P\{\chi^2 > 30.578\} = 1 - 0.01 = \boxed{0.99}$$

## 第2问：求 $D(S^2)$

设 $\chi^2 = \frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$

根据 [[笔记/数字特征与参数估计/4. 样本以及抽样分布.md|$\chi^2$ 分布的数学期望和方差]] 的性质：若 $\chi^2 \sim \chi^2(n)$，则 $D(\chi^2) = 2n$

所以：
$$D\left[\frac{(n-1)S^2}{\sigma^2}\right] = 2(n-1)$$

根据 [[笔记/数字特征与参数估计/2. 方差.md|方差的性质]] $D(CX) = C^2D(X)$：

$$\frac{(n-1)^2}{\sigma^4} D(S^2) = 2(n-1)$$

$$D(S^2) = \frac{2(n-1)\sigma^4}{(n-1)^2} = \frac{2\sigma^4}{n-1}$$

当 $n = 16$ 时：
$$D(S^2) = \boxed{\frac{2\sigma^4}{15}}$$
