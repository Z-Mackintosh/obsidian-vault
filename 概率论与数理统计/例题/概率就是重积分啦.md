# 例题
进行打靶，设弹着点 $A(X,Y)$ 的坐标 $X$ 和 $Y$ 相互独立，且都服从 $N(0,1)$ 分布，规定

- 点 $A$ 落在区域 $D_1 = \{(x,y) \mid x^2 + y^2 \leq 1\}$ 得 2 分；
- 点 $A$ 落在 $D_2 = \{(x,y) \mid 1 < x^2 + y^2 \leq 4\}$ 得 1 分；
- 点 $A$ 落在 $D_3 = \{(x,y) \mid x^2 + y^2 > 4\}$ 得 0 分。

以 $Z$ 记打靶的得分，写出 $X,Y$ 的联合概率密度，并求 $Z$ 的分布律。

# 解答

## 方法分析
本题是二维正态分布与离散型随机变量分布律的综合问题。解题关键在于：
1. 利用独立正态分布的性质求联合概率密度
2. 通过极坐标变换计算圆环区域概率
3. 建立得分与区域概率的对应关系

## 解题步骤

### Step 1：求联合概率密度
由题设，$X \sim N(0,1)$，$Y \sim N(0,1)$，且 $X$ 与 $Y$ 相互独立。

$X$ 的概率密度为：
$$f_X(x) = \frac{1}{\sqrt{2\pi}} e^{-x^2/2}, \quad -\infty < x < \infty$$

$Y$ 的概率密度为：
$$f_Y(y) = \frac{1}{\sqrt{2\pi}} e^{-y^2/2}, \quad -\infty < y < \infty$$

由于 $X$ 和 $Y$ 相互独立，联合概率密度为：
$$f(x,y) = f_X(x)f_Y(y) = \frac{1}{2\pi} e^{-\frac{1}{2}(x^2+y^2)}, \quad -\infty < x < \infty, \quad -\infty < y < \infty$$

### Step 2：计算各区域概率
区域 $D_1 = \{(x,y) \mid x^2 + y^2 \leq 1\}$ 对应得 2 分：
$$P\{(X,Y) \in D_1\} = \iint_{D_1} f(x,y)  dx  dy$$

使用极坐标变换：$x = r\cos\theta$, $y = r\sin\theta$, $dxdy = rdrd\theta$
$$= \int_0^{2\pi}  d\theta \int_0^1 \frac{1}{2\pi} e^{-r^2/2} r  dr$$
$$= 2\pi \times \frac{1}{2\pi} \left[ -e^{-r^2/2} \right]_0^1 = 1 - e^{-1/2}$$

区域 $D_2 = \{(x,y) \mid 1 < x^2 + y^2 \leq 4\}$ 对应得 1 分：
$$P\{(X,Y) \in D_2\} = \iint_{D_2} f(x,y)  dx  dy = \int_0^{2\pi}  d\theta \int_1^2 \frac{1}{2\pi} e^{-r^2/2} r  dr$$
$$= -e^{-r^2/2} \bigg|_1^2 = e^{-1/2} - e^{-2}$$

区域 $D_3 = \{(x,y) \mid x^2 + y^2 > 4\}$ 对应得 0 分：
$$P\{(X,Y) \in D_3\} = 1 - P\{(X,Y) \in D_1\} - P\{(X,Y) \in D_2\} = e^{-2}$$

### Step 3：确定分布律
根据各区域概率与得分的对应关系，$Z$ 的分布律为：

| $Z$   | 0        | 1                   | 2              |
| ----- | -------- | ------------------- | -------------- |
| $p_k$ | $e^{-2}$ | $e^{-1/2} - e^{-2}$ | $1 - e^{-1/2}$ |

最终答案：
$$\boxed{P(Z=0)=e^{-2},\quad P(Z=1)=e^{-1/2}-e^{-2},\quad P(Z=2)=1-e^{-1/2}}$$

# 核心技巧

## 概率密度计算
- 利用独立随机变量的性质：联合概率密度等于边缘概率密度的乘积
- 对于标准正态分布，要熟悉其概率密度函数的表达式

## 区域概率计算
- 对于圆形或环形区域，优先考虑极坐标变换
- 极坐标变换时注意雅可比行列式 $rdrd\theta$ 的引入
- 积分上下限要根据区域形状正确确定

## 分布律建立
- 建立离散型随机变量取值与对应事件概率的映射关系
- 验证所有概率之和为 1：$e^{-2} + (e^{-1/2}-e^{-2}) + (1-e^{-1/2}) = 1$

# 经验总结
1. **独立正态分布的性质**：两个独立的标准正态分布的联合概率密度具有旋转对称性，这为极坐标变换提供了便利。
2. **极坐标变换技巧**：当积分区域是圆形、环形或扇形时，极坐标变换能显著简化计算，关键在于正确确定 $r$ 和 $\theta$ 的积分限。
3. **概率完备性验证**：计算离散型随机变量分布律后，务必验证所有概率之和为 1，这是检验计算正确性的重要方法。

## 验证
验证概率之和：
$$e^{-2} + (e^{-1/2}-e^{-2}) + (1-e^{-1/2}) = 1$$
计算验证：
$$e^{-2} + e^{-1/2} - e^{-2} + 1 - e^{-1/2} = 1$$
验证通过，分布律正确。
