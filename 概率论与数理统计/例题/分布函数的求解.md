# 例题
已知随机变量 $X$ 的分布函数为
$$ F(x) = \begin{cases} 
0, & x < -1, \\
\frac{1}{8}, & x = -1, \\
ax + b, & -1 < x < 1, \\
1, & x \geq 1.
\end{cases} $$

已知 $P\{-1 < X < 1\} = \frac{5}{8}$，求解 $a,b$. 以及 $P\{-1 \leq X < 0\}$ 的值

# 解答
$P\{-1 < X < 1\} = P\{X < 1\}-P\{X \leq -1\}$, 则
$P\{-1 < X < 1\} = F(1^{-})-F(-1)=a+b- \frac{1}{8}= \frac{5}{8}$, 得方程组 $a+b= \frac{3}{4}$
又, 考虑到[[2.随机变量及其分布函数#^f60b40|连续性]], 得方程组
$-a+b= \frac{1}{8}$,
联立解得
$$
a= \frac{5}{16},b=\frac{7}{16}
$$


$P\{-1 \leq X < 0\}=P\{X < 0\}-P\{ X < -1\}=F(0^{-})-F(-1^{-})$
$P\{-1 \leq X < 0\}=b+0$
$$
\frac{7}{16}
$$

## 分布函数连续性核心要点

### 1. 分布函数的特有性质
**右连续性是分布函数的根本特征** - 所有分布函数都必须满足 $F(x_0) = F(x_0 + 0)$，这是由定义 $F(x) = P\{X \leq x\}$ 直接决定的。

### 2. 关键应用技巧
在分段分布函数中，**连接点处的右连续性**是建立方程的核心：

- **跳跃点的识别**：$P\{X = x_0\} = F(x_0) - F(x_0 - 0)$
- **连续段的处理**：线性段与常数段的连接必须满足右连续
>  等号跟大于号走，永远右连续
### 3. 本题关键步骤
**在 $x = -1$ 处利用右连续性**：
- $F(-1) = \frac{1}{8}$（给定值）
- $F(-1 + 0) = \lim_{x \to -1^+}(ax + b) = -a + b$ 
- **右连续性要求**：$-a + b = \frac{1}{8}$（这是解方程的关键！）

### 4. 快速解题模式
对于含未知参数的分段分布函数：
1. **找连接点**：分段函数的交界点
2. **用右连续**：$F(x_0) = \lim_{x \to x_0^+} F(x)$
3. **配概率条件**：$P\{a < X < b\} = F(b^-) - F(a)$
