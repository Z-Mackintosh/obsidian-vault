# 第一部分：代码运行及其解释
## 第一题
- 源代码
```cpp
#include <iostream>

int main() {
    char i, j;
    i = 'A';
    j = 'B';
    std::cout << i << ", " << j << ", " << i + j << std::endl;
    return 0;
}
```
- 运行结果
$$
A，B，131
$$
- 解释
	1. 定义变量 i, j.赋值字符 A 和 B
	2. 打印部分分为两部分，一部分是打印出字符 A 和 B
	3. 另一部分通过运算符号连接 i, j 变量，计算其 ASCII码，前者为 65，后者为 66 ，故结果为 131.
## 第二题
- 源代码
```cpp
#include <iostream>

int main(){
    int i,j;
    int m,n;
    i = 8; j = 10;
    m = ++i + j++;
    n = (++i) + (++j) + m;
    std::cout << i << "," << j << std::endl;
    std::cout << m << "," << n << std::endl;
    return 0;
}
```
- 运算结果 
$$
10,12 
$$
$$
19,41
$$
- 解释
	1.  赋值变量 i, j 为 8 和 10
	2. 最终结果输出 i, j，观察代码可知，i 经历了两次自增后返回，结果为 10；j 经历一次返回后自增和自增后返回，结果为 12
	3. 最终结果输出 m, n, 观察代码可知，m 为 i 自增后返回加上 j 返回后自增的值相加，为 19，n 为 i 和 j 在经历一次自增后返回的值再与 m 相加后的值，为 42。
